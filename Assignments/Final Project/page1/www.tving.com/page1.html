자주 찾는 질문

  • TVING 회원가입은 TVING ID, SNS ID, CJ ONE ID를 통해 가입이 가능합니다.

    ■ 회원가입 ID 종류
    1. TVING ID : TVING에서만 사용하는 TVING 전용 ID
    2. SNS ID : 사용하시는 SNS (네이버, 카카오, 페이스북, 트위터, Apple)의 계정을 연동하여 가입하는 ID
    3. CJ ONE ID : CJ ONE 통합 멤버십 ID

    ■ 회원가입 방법
    1. WEB : 티빙 시작하기 → TVING 아이디, CJ ONE 아이디, SNS 아이디 (네이버, 카카오, 페이스북, 트위터, Apple) 중 1개 선택 → 회원가입
    2. APP : 티빙 시작하기 → TVING 아이디, CJ ONE 아이디, SNS 아이디 (네이버, 카카오, 페이스북, 트위터, Apple) 중 1개 선택 → 회원가입

  • TVING WEB과 APP은 아래와 같은 방법으로 로그인이 가능합니다.

    ■ TVING 로그인 방법
    1) 티빙 WEB/APP 접속
    2) '티빙 시작하기' 버튼 클릭
    3) 계정 선택화면에서 회원가입하신 계정 유형 선택
    4) 아이디, 비밀번호 입력 후 '로그인하기' 버튼 클릭

    혹시 일치하는 회원정보가 없다는 알림 메시지가 나오신다면 아래 사항을 확인하여 주세요.

    ■ TVING 로그인 안될 시 조치 방법
    1) WEB 브라우저 또는 APP 좌측 상단의 '뒤로가기'를 클릭하여 계정 유형 선택 화면으로 이동
    2) 회원가입한 유형을 다시 확인하여 선택
    - CJ ONE 통합회원이신 경우 'CJ ONE으로 시작하기' 선택 (제일 밑에 위치)
    - TVING ID로 가입하신 경우 'TVING ID로 시작하기' 선택
    - 네이버, 카카오 등 SNS 연동 계정으로 가입하신 경우 '각 SNS로 시작하기' 선택
    3) 아이디, 비밀번호 입력 후 '로그인하기' 버튼 클릭하여 로그인

    * 'TVING ID'로 로그인 시도하셨는데 일치하는 회원정보가 없다면 먼저 'CJ ONE으로 시작하기'를 선택하여 확인을 부탁드립니다.
    * 아이디가 이메일 형태의 계정인데 'TVING ID'로 로그인이 안되시는 경우 SNS 연동 회원일 수 있으며, 네이버, 카카오 등 '각 SNS로 시작하기'를 선택하여 확인을 부탁드립니다.
    * 계정 유형을 맞게 선택하신 경우 '아이디 찾기', '비밀번호 찾기'를 진행해 주세요.

    지속해서 로그인이 되지 않으시는 경우 1:1 게시판 문의 또는 tving@cj.net 으로 문의해 주시면,
    신속하게 가입하신 계정 확인하여 답변드리겠습니다.

  • 티빙 이용권은 베이직, 스탠다드, 프리미엄 이용권으로 구성이 되어 있으며,
    네이버 플러스 멤버쉽 가입 후, 디지털 서비스에서 티빙을 선택하시면 이용하실 수 있는 방송 무제한 이용권도 준비되어 있습니다.

    이용권 별 혜택은 다음과 같습니다.

    ※ 티빙 이용권
    1) 베이직 이용권
    - 실시간 + TV프로그램 + 영화(개별구매 제외), 720p HD 화질 이용 가능 (모바일 + PC)
    - 동시 시청 1명

    2) 스탠다드 이용권
    - 실시간 + TV프로그램 + 영화(개별구매 제외), 1080p FHD 화질 이용 가능 (모든 디바이스)
    - 동시 시청 2명

    3) 프리미엄 이용권
    - 실시간 + TV프로그램 + 영화(개별구매 제외), 1080p FHD(모든 디바이스) + 4K 화질(스마트TV 일부 콘텐츠 한정) 이용 가능
    - 동시 시청 4명

    4) 방송 무제한 (네이버 플러스 멤버십)
    - 실시간 + TV 프로그램 + T ONLY 콘텐츠 시청, 720p HD 화질 이용 가능
    - T ORIGINAL 및 월정액 영화 이용 불가
    - 동시 시청 1명

  • 정기결제란, 1개월 또는 1년 단위로 정기적으로 자동 결제가 이루어지는 편리한 서비스입니다.
    최초 결제시 등록하신 결제수단을 통해서 서비스를 계속 이용하실 수 있도록 정기적으로 결제가 이루어지기 때문에 매월/매년 다시 결제하는 번거로움 없이 지속적으로 티빙 서비스를 이용하실 수 있습니다.

    단, 정기결제 이용중 최초 결제시 등록하신 핸드폰/신용카드/네이버페이의 결제 정보가 변경될 경우 서비스가 중지될 수 있으니 참고해 주시기 바랍니다.

  • T ORIGINAL은 TVING에서 자체 제작한 콘텐츠를 의미합니다.

    주요 콘텐츠는 많은 고객님들께서 사랑해주셨던 [환승연애], [술꾼도시 여자들], [신서유기 스페셜 스프링캠프], [여고추리반] 등이 있으며, T ORIGINAL 콘텐츠의 시즌2 및 신규 콘텐츠를 준비 중에 있으니 앞으로 더 재미있는 티빙을 기대해주시고 많은 이용 부탁드립니다.

  • T ORIGINAL 콘텐츠는 '베이직, 스탠다드, 프리미엄 이용권'을 보유하셔야 시청이 가능합니다.

    네이버플러스 멤버십 방송 무제한 이용권의 경우, ORIGINAL 콘텐츠와 영화 시청이 불가하므로 아래 방법을 참고하시어 베이직, 스탠다드, 프리미엄 이용권으로 업그레이드 신청 후 이용해 주시길 바랍니다.

    ■ 네이버플러스 멤버십 이용권 업그레이드 방법
    1) PC : TVING WEB → 상단 '프로필 아이콘' → 이용권 → 변경하고자 하는 이용권 선택 → 이용권 변경하러 가기
    2) APP : TVING APP → 상단 '프로필 아이콘' → 네이버플러스 멤버십 변경 → 이용권 변경하러 가기
    3) 네이버플러스 멤버십 : 네이버플러스 멤버십 MY 페이지 → 디지털 콘텐츠+ → TVING 업그레이드신청

    * 업그레이드 이용권 선택 후 정기결제 진행 시, 즉시 혜택 변경됩니다.

    추가적으로 T ORIGINAL 콘텐츠의 경우 개별구매로 시청이 불가하오니, TVING 베이직, 스탠다드, 프리미엄 이용권 구매 후 많은 이용 부탁드립니다.

  • 모바일 상품권은 11~12자리의 숫자로 구성되며, 쿠폰 등록 메뉴에서 등록하실 수 있습니다.

    [모바일 상품권 등록 경로]
    ① PC : TVING WEB → 로그인 → 상단 '프로필 아이콘' → 쿠폰등록 → 상품권 번호 등록
    ② APP : TVING APP → 로그인 → 상단 '프로필 아이콘' → 쿠폰등록 → 상품권 번호 등록

    단, iOS 기기에서는 APP 내 모바일 상품권 및 쿠폰 등록 메뉴를 제공하지 않아, 사파리 등 브라우저에서 TVING 모바일 웹 진입 후 최하단 '쿠폰 등록하기' 메뉴를 통해 등록하실 수 있습니다.

    혹시 이미 티빙 상품을 결제하여 이용중이신 경우 모바일 상품권을 추가 등록하실 수 없으며, 현재 사용중인 이용권의 이용기간 만료 후 모바일 상품권을 등록하여 이용하실 수 있습니다.
